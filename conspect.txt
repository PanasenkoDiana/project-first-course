tilemap - карта уровня которая состоит из тайтлов
Тайт - это одна небольшая часть игры (чаще всего квадратная)

Тайлмапы не рисуются в paint или фотошопе потому-что они должны быть
сохранены в специальном формате .tmx
Для рисования тайлмапов мы используем специальную программу - tiled
Ее нужно скачивать отдельно!

Обязательно нужно сохранить tileset в папку картинок

Чтобы отображать тайлмапы мы используем специальную библиотеку pytmx
Его нужно установить отдельно с помощью терминала pip install pytmx

Унаследование классов - это способ создания новых классов на базе уже существующих
Класс родитель - это класс который будет наследоваться
Дочерний класс - это класс который будет наследовать родителя
